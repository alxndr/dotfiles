# Named AI tells — patterns to avoid

| Name | Pattern to avoid | Example of the wrong thing |
|---|---|---|
| **Hedge Stack** | Layered qualifications | "While it is important to note that X, it should be acknowledged that Y" |
| **Preamble Promise** | Announcing what you're about to say | "In this response, I'll walk you through my experience with..." |
| **Transition Summary** | Summarizing before moving on | "Now that we've covered X, let's turn to Y." |
| **Enthusiasm Injection** | Hollow exclamations | "This is really exciting!", "I'm passionate about..." |
| **Nested Qualification** | Over-hedged opinions | "there may be cases where X could potentially be considered..." |
| **Corporate Verb** | Jargon verbs | "leverage", "utilize" (vs "use"), "synergy", "deliverable" |
| **Passive Distancer** | Impersonal constructions | "It is believed that...", "It should be noted that..." |
| **Closer Summary** | Summarizing at the end | "In conclusion...", "To summarize...", "In summary..." |
| **Delve/Dive** | AI-standard transition verbs | "Let's delve into...", "Let's dive deep into..." |
| **Realm Of** | Vague domain-framing | "in the realm of web development...", "in the space of..." |
| **The Landing** | Confident-sounding but empty close | "...and that's what makes this role such an exciting opportunity." |
| **Pivot Closer** | "[Long sentence]. That's the [short restatement]." | "CompanyName does X. That's the work I want to be doing." |
| **Em-Dash Itemizer** | Packing a keyword list into an em-dash aside to signal completeness | "My primary stack — Node.js, TypeScript, React, AWS — covers everything you listed." |
| **Exact Match Claim** | "X is exactly what [company/role] [wants/uses/needs]" — asserting fit rather than demonstrating it | "My background is exactly what CompanyName is looking for." / "...is exactly what CompanyName uses." |
| **Reality Reframe** | Using "actual", "real", or "really" to frame an obvious conclusion as a revelation | "...was the actual job." / "That's what it's really about." / "That's the real challenge." |
| **Gerund Pedestal** | "(Gerund phrase) is something I (verb)" — distances the subject and sounds stilted | "Working without a PM is something I'm not afraid of." / "Collaborating cross-functionally is something I enjoy." |
